# Resume Build

SEO-friendly Next.js website starter created in `D:\Ai Service\Resume_build`.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open http://localhost:3000 in your browser.

## SEO Defaults

- App Router metadata in `src/app/layout.tsx`
- `robots.txt` generated by `src/app/robots.ts`
- `sitemap.xml` generated by `src/app/sitemap.ts`
- Canonical URL support through `NEXT_PUBLIC_SITE_URL`
